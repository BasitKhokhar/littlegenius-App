## 🚀 Quick Start

### 1. Install Dependencies
```bash
cd "D:\Codes\Apps\LittleGenius App"
npm install
```

### 2. Test Locally
```bash
npm start
npm run android
```

### 3. Build for Play Store
```bash
npm install -g eas-cli
eas login
eas build -p android --release
```

### 4. Upload to Play Store
Follow **BUILD_AND_DEPLOY.md** for detailed instructions
